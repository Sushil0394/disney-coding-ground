What is Coding ?

Coding is a set of instructions for a computer to perform an operation using programming language.

Use of coding ?

Coding is used to create Apps, Websites, Software, etc..

Examples of Programming Language ?

  • C
  • C++
  • Java
  • Python

What is HTML ?

  • HTML stands for Hypertext Markup Language.
  • It is a text based markup language for documents designed to be displayed in a web browser.
  • Consists of head, body and footer section.
  • It is a case insensitive language.
  • HTMl files are saved with an extension of .html
  • It starts with a start tag/opening tag and ends with an end tag/closing tag.

What is CSS ?

  • CSS stands for Cascading Style Sheet
  • CSS describes how HTML elements are to be displayed on screen

Types of Style-Sheet ?

  • Inline style-sheet.
  • Internal style-sheet.
  • External style-sheet.
